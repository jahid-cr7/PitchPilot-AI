## How to Use

1. **Preparing for an interview?** Read [`INTERVIEW_PITCH.md`](INTERVIEW_PITCH.md) and [`PROJECT_SUMMARY.md`](PROJECT_SUMMARY.md).
2. **Recording a demo video?** Follow [`DEMO_RECORDING_CHECKLIST_V1_1.md`](DEMO_RECORDING_CHECKLIST_V1_1.md) and read [`DEMO_VIDEO_SCRIPT_V1_1.md`](DEMO_VIDEO_SCRIPT_V1_1.md).
3. **Posting on LinkedIn?** Copy from [`LINKEDIN_POST_V1_1.md`](LINKEDIN_POST_V1_1.md).
4. **Publishing a GitHub release?** Paste [`GITHUB_RELEASE_DESCRIPTION_V1_1.md`](GITHUB_RELEASE_DESCRIPTION_V1_1.md) into the release notes.
5. **Updating your CV?** Copy bullets from [`RESUME_BULLETS.md`](RESUME_BULLETS.md).
